# Portfolio Website Navigation Overview

This document outlines the navigation structure of my personal portfolio website, detailing each page and its primary sections.

---

## Main Navigation Menu

The website's main navigation menu is accessible from all pages and includes the following links:

1. **Home** – Landing page with introduction
2. **Skills** – My technical skills and tools
3. **Projects** – Showcase of my work
4. **Experience** – Professional background
5. **Contact** – Get in touch with me

---

## Detailed Page Structure

### 1. Home

**Sections:**

- Hero introduction with name and role (Frontend Engineer)
- Quick skills highlights
- Featured projects teaser
- Experience summary
- Call-to-action to view full projects or contact me

### 2. Skills

**Sections:**

- Technical Skills: Detailed list of technologies I use daily (React, Next.js, TypeScript, Tailwind CSS, React Query, Framer Motion, etc.)
- Tools & Soft Skills

### 3. Projects

**Sections:**

- All personal and professional projects with:
  - Short description
  - Tech stack used
  - Live demo link
  - GitHub repository link
  - Screenshots

Featured projects include:

- Portfolio Website 2026 (current site)
- X Clone App
- Anime List App
- Previous Portfolio 2025
- SaaS Landing Page
- WordPress Travel Website

### 4. Experience

**Sections:**

- Professional Experience (IT Support & Check-in at PT Jasa Angkasa Semesta)
- Personal Projects (treated as real development experience)

### 5. Contact

**Sections:**

- Contact form (name, email, message)
- Direct links to email, LinkedIn, and GitHub
- Quick response promise

---

## Additional Features

- **Theme Toggle** (Light ↔ Dark mode)
- **Fully Responsive** (works perfectly on mobile, tablet, and desktop)
- **Fast Loading** (optimized with Next.js 15)
- **Modern Animations** (powered by Framer Motion)
