# Pande Deva - Frontend Engineer Portfolio

Personal portfolio website built with **Next.js 15**, React, TypeScript, Tailwind CSS, and shadcn/ui.

## Tech Stack

- Next.js 15 (App Router)
- React 19 + TypeScript
- Tailwind CSS + shadcn/ui
- React Query + Framer Motion
- Vercel Deployment
